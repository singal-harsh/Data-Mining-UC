Let’s first load the Iris dataset. This is a very famous dataset in almost all data mining, machine learning courses, and it has been an Python's sklearn build-in dataset. The dataset consists of 50 samples from each of three species of Iris flowers (Iris setosa, Iris virginicaand Iris versicolor). Four features(variables) were measured from each sample, they are the length and the width of sepal and petal, in centimeters. It is introduced by Sir Ronald Fisher in 1936.

  • 3 Species

  • Four features of flower: length and the width of sepal and petal

2.1 Histogram¶

Histogram is the easiest way to show how numerical variables are distributed.

2.1.0.1 Produce a single histogram¶

In [28]:
plt.figure(figsize = (8,6))

# Alternate - plt.hist(iris['sepal length (cm)'], bins = 20, density = True)

plt.ylabel('Frequency')
plt.show()

You may change “bins=” and “color=” to have different appearance.

Density plot is a nonparametric fitting

You can make the plot more elegant with different options. For example, adding a title, adjusting the axis range, renaming the axis label, and so on…

4.1 K-Nearest Neighbor (KNN)¶
